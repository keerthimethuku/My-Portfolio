Food Delivery App is a full-stack web application built using the MERN stack that allows users to browse food items, place orders, and manage their order status in real-time. The frontend was developed with React and styled using Bootstrap for a clean, responsive user interface. Dynamic routing was handled with React Router, enabling smooth navigation between pages. The backend was implemented using Node.js and Express, with RESTful APIs managing user authentication, food item listings, and order processing. Notes App

Notes App is a full-stack, responsive web application built using the MERN stack, enabling users to create, edit, delete, and manage personal notes with data persistence. The backend was implemented with Node.js, Express, and MongoDB to securely store and retrieve note data via RESTful APIs. The application features a clean, responsive user interface and efficient data handling to deliver a seamless note-taking experience across devices.
